# Security Policy

## Supported Versions
Only the latest version of this architecture is supported for security updates.

## Reporting a Vulnerability
I am committed to infrastructure security. If you discover a configuration drift or a vulnerability in this IaC implementation, please open a GitHub Issue or contact me directly via the details on my profile.

**Current Posture:** 
- Automated Static Analysis (Trivy) integrated into CI.
- Mandatory encryption-at-rest for all persistence layers.
